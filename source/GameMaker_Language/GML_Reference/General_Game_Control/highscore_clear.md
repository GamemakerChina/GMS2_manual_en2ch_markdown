# highscore_value

With this function you can clear the internal global high score list,
resetting all names to the default value of "Unknown" and all scores to
the default value of 0.

#### Syntax:

``` gml
highscore_clear();
```

#### Returns:

``` gml
N/A
```

#### Example:

``` gml
highscore_clear();
```

This would clear the high score list.
