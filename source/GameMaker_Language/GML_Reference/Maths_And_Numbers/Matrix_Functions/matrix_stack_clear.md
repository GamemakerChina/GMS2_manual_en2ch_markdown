# matrix_stack_clear

This function clears the matrix stack.

#### Syntax:

``` gml
matrix_stack_clear();
```

#### Returns:

``` gml
N/A
```

#### Example:

``` gml
if !matrix_stack_is_empty()
{
    matrix_stack_clear();
}
```

The above code checks the matrix stack to see if it is empty and if it
is not it clears it.
